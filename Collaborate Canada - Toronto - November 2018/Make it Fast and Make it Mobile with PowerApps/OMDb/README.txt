1. Go to http://www.omdbapi.com/apikey.aspx and register for an API key
2. Open OMDb.swagger.json and replace "MYKEY" with your API key and save
3. In your PowerApps environment -> Custom Connectors -> Create custom connector -> Import an OpenAPI file
4. Choose OMDb.swagger.json
5. In your PowerApps environment -> Apps -> Import Package
6. Choose OMDbSearch_20181120043914.zip
7. OMDb (connection) -> Select during import 
8. Choose OMDb
9. Import
